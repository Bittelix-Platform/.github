## 🚀 About Us

**Bittelix** is a trading ecosystem designed to make automated trading more accessible and transparent. It combines AI-assisted automation, a professional strategy marketplace, a secure P2P exchange, and a free AI analysis assistant — all in one platform. Built for crypto and forex markets, Bittelix helps traders of every level focus on what matters: smarter decisions and better risk management.

## 🔒 Security & Privacy

- **Private repository** – core code is not publicly accessible
- **No sensitive data** in the repo — API keys and secrets are managed as environment variables
- **Trade-only API access** — withdrawal permissions are never required
- **3-layer escrow** planned for P2P transactions
- All communications encrypted (TLS 1.3)
